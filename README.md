**Project Title:**

Video Game Sales Analysis (1980–2024)



**Objective:**

Analyze global video game sales to identify trends, regional preferences,

top-performing genres, consoles, and publishers using Excel.



**Dataset:**

Video game sales dataset covering title-level sales by region.



**Key Fields:**

\- Name

\- Platform

\- Genre

\- Publisher

\- Year

\- NA\_Sales

\- JP\_Sales

\- PAL\_Sales

\- Other\_Sales

\- Global\_Sales

\- Critic\_Score



**Data Cleaning Steps:**

\- Standardized missing values

\- Removed invalid year entries

\- Created helper columns for regional sales ratios

\- Ensured numeric consistency for sales fields



**Key Business Questions:**

1\. How have global video game sales evolved over time?

2\. Which genres perform best globally and by region?

3\. How do regional preferences differ across NA, JP, and PAL?

4\. Which consoles dominate global sales?

5\. Is there a relationship between critic score and total sales?

6\. Which publishers consistently produce high-selling games?



**Sheet Guide:**

\- Raw\_Data        → Original dataset

\- Cleaned\_Data    → Cleaned and standardized data

\- Helper\_Columns  → Calculated ratios and indicators

\- Pivot\_Analysis  → Pivot tables used for analysis

\- Charts          → Final visualizations

\- Insights\_Summary→ Written insights per question
